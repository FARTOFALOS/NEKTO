- identity, meaning, coherence, and causality are treated as architectural strata rather than loose metaphors;
- reading order matters;
- awareness of available faculties is not the same as simultaneous activation of all faculties;

For a human reader, this repository may look like a hybrid of ontology, architecture, and philosophical systems design.
